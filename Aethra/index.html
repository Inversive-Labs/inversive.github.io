Aethra

Security intelligence platform for Solana programs. LLM-powered vulnerability database with historical sBPF bytecode analysis, execution flow pattern detection, and AI-guided security audits.

Overview

Solana vulnerability database indexing historical programs, exploits, and security patterns. Combines traditional indexing with LLM-powered analysis to detect vulnerabilities, identify execution flow patterns, and conduct automated security audits.

Query historical sBPF bytecode, analyze attack patterns, detect function flow sequences, and leverage AI agents for intelligent security research.
